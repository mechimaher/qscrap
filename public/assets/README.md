# QScrap Website Assets

This directory contains images for the official QScrap website.

## Required Images

1. **hero-car-parts.jpg** - Hero section image showing premium car parts (suggested: 1200x800px)
2. **partner-garage.jpg** - Partner section image showing a modern garage (suggested: 800x600px)
3. **og-image.jpg** - Open Graph social sharing image (suggested: 1200x630px)
4. **apple-touch-icon.png** - iOS home screen icon (180x180px)

## Image Guidelines

- Use premium, high-resolution automotive/industrial imagery
- Maintain Qatar/Maroon/Gold color theme where possible
- Prefer images with modern, clean aesthetics
- Avoid cluttered or low-quality stock photos

## Sourcing

Recommended sources for royalty-free images:
- Unsplash (automotive, industrial)
- Pexels (car parts, garages)
- Adobe Stock (premium option)
